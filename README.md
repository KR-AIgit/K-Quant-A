# 🌌 NEURAPEX AI (뉴라펙스 AI) — 실시간 지능형 가상화폐 시황 대시보드

> **"무릎에 사서 어깨에 파는" 인공지능 트레이딩 설계를 다크 글래스모피즘 계기판에 담아내다.**

NEURAPEX AI는 국내 양대 암호화폐 거래소인 **빗썸(Bithumb)**과 **코인원(Coinone)**의 공개 API를 실시간으로 교차 분석하여, 아비트라지 차액(김치프리미엄)과 시세 모멘텀을 직관적으로 시각화하고 최적의 기대 손익비 거래 신호를 실시간으로 계산하는 **미래지향적 AI 트레이딩 보조 대시보드**입니다.

---

## ⚡ 핵심 기능 (Key Features)

### 1. 🎯 [AI 트레이딩] 신경망 의사결정 시그널 보드
* **무릎-어깨 전략 실현**: 시장의 극단적 공포(발바닥 투매)와 탐욕(머리끝 매도)을 지양하고, 가장 안정적인 **"무릎 진입, 어깨 익절, 발바닥 손절"** 기대 손익비 범위를 실시간 시세 대비 백분율 가중치로 자동 연산합니다.
* **시간대별 가중치 연동**: 사용자가 우측 차트 필터(**30분, 1시간, 24시간, 7일, 전체** 등)를 클릭하는 순간, AI 엔진이 시간 간격에 따른 거래 성격(초단기 스캘핑, 스윙, 장기 매집 등)을 파악하여 **진입/목표가/손절 밴드를 실시간으로 재구성**합니다.
* **한국어 AI Rationale (의사결정 근거)**: 양대 거래소의 실시간 호가 매수/매도 벽 잔량 비율과 거래대금 가중평균가(VWAP) 데이터를 전이학습 모델로 스캔하여, 정교하게 추론된 한국어 근거 분석을 매칭 제공합니다.

### 2. 📊 실시간 가격 비교 및 아비트라지 계산기
* **실시간 가격 수집**: 빗썸과 코인원 API를 동시 크롤링하여 양대 거래소의 현재가를 5초(또는 시뮬레이터 1.5초) 주기로 갱신하며 시세 변화 시 영롱한 네온 플래시(상승: 초록, 하락: 빨강) 효과를 부여합니다.
* **스프레드 자동 계측**: 두 거래소 간의 가격 차액과 프리미엄 비율(김프)을 실시간 소수점 3자리까지 환산하여 매수/매도 기회의 유리한 방향성을 직관적으로 표기합니다.
* **종합 시장 통계**: 24H 최고가/최저가, 변동률 및 조 원/억 원 단위를 한글로 자동 변환하는 지능형 거래대금 통계 판넬을 제공합니다.

### 3. 📈 구간 인지형(Interval-aware) 실시간 네온 차트
* **타임라인 왜곡 방지**: 틱당 시그널 유입 시 그래프가 좌측으로 무작정 밀려나 타임라인이 축소되던 문제를 보완하고, **선택된 시간 간격(30분/1시간 차트는 1분 주기, 24시간 차트는 1시간 주기)이 실제로 도래할 때만 데이터를 밀어내는(Shift) 스마트 틱 메커니즘**을 구현했습니다.
* **실시간 Ticking**: 시간 축이 정밀하게 보존되는 와중에도 실시간 시세 변동이 발생하면 차트의 가장 마지막 포인트가 위아래로 역동적으로 움직여 살아있는 UX를 전해줍니다.
* **글로우 네온 그라데이션**: 보라색과 청록색 네온 그라데이션이 적용되어 다크모드 대시보드의 미래지향성을 배가시킵니다.

### 4. 🧠 진짜 AI 스마트 애널리스트 리포트
* 등락폭, 아비트라지 편차, RSI 기술 지수를 동적 스캔하여 실시간 인공지능 종합 판독 보고서 및 매매 신호(`STRONG BUY / BUY / HOLD / SELL / STRONG SELL`)를 실시간 출력합니다.
* AI 온체인 지표 검진 탭을 통해 고래(Whale) 매집도, MVRV Z-Score, 네트워크 보안 해시레이트 등 심층 데이터 브리핑을 제공합니다.

### 5. 💬 localStorage 연동 영구 보존 토론 게시판
* 사용자의 닉네임, 투자 의견(상승/중립/하락), 코멘트 내용을 브라우저의 `localStorage`에 즉시 저장하여 새로고침을 하거나 창을 닫아도 **영구적으로 대화 이력이 복원**됩니다.
* 코인을 전환할 때마다, 커뮤니티의 소통 분위기를 돋우기 위해 해당 코인의 기술적 특징에 부합하는 **인공지능 분석원 봇 및 가상 전문가들의 시황 예측 선제 댓글(AI Expert Thoughts)**을 자동으로 소환하여 꽉 찬 커뮤니티 감성을 완성합니다.

### 6. 🌐 Robust Fallback (무중단 시뮬레이터 시스템)
* 외부 API 호출 오류 또는 네트워크 단절, CORS(교차 출처 제한) 필터 오동작에 대비하여 다중 프록시(CORS Proxy) 체인을 지원하며, 네트워크 차단 감지 시 즉시 **정교한 가격 확률 분포 시뮬레이터(Random Walk) 모드로 자동 전향**되어 365일 무중단 동작을 보장합니다.

## 🛠️ 기술 스택 (Tech Stack)

* **Core**: Semantic HTML5, Vanilla JavaScript (ES6+)
* **Styling**: Vanilla CSS3 (다크 글래스모피즘, 네온 빛 잔상 애니메이션, 쉬머 테두리 글로우 효과)
* **Libraries & APIs**:
  * [Chart.js (v4.x)](https://www.chartjs.org/) - 실시간 광원 렌더링 그래프 구현
  * Bithumb Public Ticker API
  * Coinone Public Ticker API
  * CORS Proxy Solutions (corsproxy.io, allorigins)

## 🚀 실행 및 배포 방법 (Installation & Run)

### 로컬 환경에서 실행하기
NEURAPEX AI는 **서버 설치나 복잡한 의존성 빌드 과정이 전혀 없는 순수 프론트엔드 정적 웹 애플리케이션**입니다.

1. 본 저장소의 모든 파일(`index.html`, `styles.css`, `app.js`)을 다운로드하거나 로컬 폴더에 복사합니다.
2. 폴더 내의 **`index.html`** 파일을 더블클릭하여 사용하는 웹 브라우저(Chrome, Safari, Edge 등)로 즉시 실행합니다.

### GitHub Pages를 통해 무료 호스팅 배포하기
이 리포지토리를 GitHub에 업로드하면 **GitHub Pages 서비스를 통해 무료로 실시간 웹 URL로 배포**하여 전 세계 어디서든 모바일이나 웹으로 즉시 접속할 수 있습니다.

1. GitHub 계정에 새 저장소(Repository)를 생성합니다.
2. 본 소스 파일들을 업로드합니다.
3. 저장소의 **[Settings] -> [Pages]** 탭으로 이동합니다.
4. Build and deployment의 Branch를 `main` (또는 `master`)으로 선택하고 `/ (root)`를 지정한 뒤 **[Save]** 버튼을 누릅니다.
5. 약 1~2분 뒤 생성되는 고유 웹 링크(`https://[사용자ID].github.io/[저장소명]/`)를 통해 온라인상에서 자유롭게 접속할 수 있습니다!
